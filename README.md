## 프로젝트 소개

- 모바일 앱을 통해 사람들에게 선호 스타일을 통해 패션 코드(FTTI)를 부여합니다.
- FTTI를 바탕으로 옷 스타일에 대한 큐레이션과 추천을 해줍니다.

- Code Editor: VS Code, Android Studio
- Collaboration Tool: Notion, Github, Slack, Figma, Google Drive
